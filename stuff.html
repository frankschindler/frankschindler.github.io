Mathematica

Here you can find some Mathematica notebooks related to non-interacting fermionic topological phases. None of this is code is optimal. I tend to minimize the lifetime I spend on coding rather than the runtime a machine spends on executing. (This is arguably not an acceptable position to take in the case of problems which scale exponentially with physical system size. For the polynomial-in-system-size simulation of non-interacting electrons it is a luxury one can afford.)

BiTB.nb
Implementation of the bismuth tight-binding model from Electronic structure of the semimetals Bi and Sb, together with a calculation of the Z4 symmetry indicator invariant that establishes bismuth as a higher-order topological insulator.

SnTeTB.nb
Implementation of the tin telluride tight-binding model from Topological crystalline insulators in the SnTe material class. Also incluces a discrete Fourier transform to 3D real-space that allows for the model to be studied in presence of open boundary conditions or crystal defects.

SmoothGaugeSSH.nb
Implementation of the parallel transport prescription from Smooth gauge for topological insulators for a time-reversal symmetric generalization of the SSH model. Also contains a gauge-invariant method for calculating the Wilson loop topological invariant.

MirrorChern2Model.nb
Realization of a two-dimensional inversion-symmetric higher-order topological insulator via mirror symmetry breaking in a topological crystalline insulator with mirror Chern number 2. Also contains a real-space calculation of the resultant corner modes.

NestedWilson.nb
Calculation of the nested Wilson loop from Quantized electric multipole insulators in a model obtained from stacking two topological insulators and gapping out their combined Wilson spectrum. Interestingly, whether the resultant nested Wilson loop is trivial or non-trivial does not depend on the details of the gap but solely on the bulk topology!
